# Ephemeral Cloud Risk Detection

## Feature 1: Asset & Identity Discovery

This module discovers and classifies ephemeral cloud resources and identities from:

- Cloud Audit Logs
- Kubernetes Events
- Identity Session Logs

### Capabilities

- Asset Discovery
- Identity Discovery
- TTL Tracking
- Lifecycle Analysis
- Ephemeral Classification
- Persistent Classification
- Ambiguous Scenario Detection

### Sample Scenarios

- Public VM Detection
- Debug Privileged Pod Detection
- Off-Hours AssumeRole Activity
- Novel Actor Detection

### Tech Stack

- Python
- Pandas
- NumPy
